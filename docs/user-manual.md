## 一、全局功能

### 1.1 顶部导航栏

页面顶部固定的毛玻璃导航栏，包含以下元素：

| 元素 | 位置 | 功能 |
|------|------|------|
| **Logo + 站名** | 左侧 | 点击返回首页 `/` |
| **保研信息 ▼** | 导航区 | 悬浮展开下拉菜单：信息聚合、院校库、导师库、竞赛信息、期刊会议 |
| **AI 辅导 ▼** | 导航区 | 悬浮展开：简历工坊、择校推荐、导师推荐、模拟面试、心理支持、综合规划 |
| **知识库 ▼** | 导航区 | 悬浮展开：院校百科、录播课程、面试题库、文书模板、经验精选、信息差速递 |
| **进度中心** | 导航区 | 直接跳转到 `/progress` |
| **社群** | 导航区 | 直接跳转到 `/community` |
| **搜索栏** | 右侧 | 显示「搜索 Ctrl+K」，点击打开全局搜索面板 |
| **主题切换** | 右侧 | 调色板图标，点击打开主题设置面板 |
| **用户头像** | 右侧 | 登录后显示，悬浮展开用户菜单 |
| **登录按钮** | 右侧 | 未登录时显示，跳转到登录页 |

**移动端**：导航菜单收起为汉堡菜单按钮，点击展开全屏导航面板。

### 1.2 全局搜索（⌘K / Ctrl+K）

按下 `Ctrl+K`（Mac: `⌘K`）或点击导航栏搜索按钮，弹出全屏搜索面板：

- **搜索框**：实时过滤，输入关键词匹配页面名称
- **分组显示**：保研信息（5 项）、AI 辅导（5 项）、社区、个人
- **键盘导航**：↑↓ 选择、回车打开、ESC 关闭
- **点击遮罩关闭**

### 1.3 主题设置面板

点击导航栏调色板图标打开，包含：

#### 颜色主题（5 种）
| 主题 | 效果 |
|------|------|
| 清新蓝 | 默认主题色，偏蓝色调 |
| 活力橙 | 暖色调橙色 |
| 学术紫 | 紫色学术感 |
| 自然绿 | 清新绿色 |
| 玫瑰粉 | 柔和粉色 |

点击色块即时切换，全站颜色随之变化。

#### 外观模式（3 种）
- **浅色**：白色背景
- **深色**：深色背景
- **跟随系统**：随操作系统暗色模式自动切换

#### 基础背景（7 种）
| 背景 | 效果 |
|------|------|
| 无 | 纯色背景 |
| 渐变 | 4 色缓慢流动渐变 |
| 极光 | 3 个彩色光球缓慢移动 + 色相旋转 |
| 波浪 | 底部 SVG 波浪起伏 |
| 星空 | 60 颗星点闪烁 |
| 粒子 | 18 个彩色粒子缓慢上浮 |
| 几何 | 12 个节点 + 连线缓慢旋转 |

#### 天气背景（10 种）
| 背景 | 效果 |
|------|------|
| 自动跟随 | 通过浏览器定位 + Open-Meteo API 获取实时天气，自动切换 |
| 晴天 | Canvas 太阳光晕 + 镜头光斑 + 底部暖色辉光 |
| 多云 | CSS 云朵纹理水平飘动 |
| 雨天 | Canvas 梯形雨滴下落 + 溅水粒子 + 云朵叠加 |
| 雷暴 | 密集斜雨 + 递归分支闪电 + 全屏白色闪光 + 浓云 |
| 雪天 | Canvas 雪花正弦摇摆 + 底部雪堆积 |
| 雾天 | CSS 烟雾纹理漂移 + 径向渐变白灰蒙版 + Canvas 雾气团 |
| 冰雹 | 六边形冰雹快速下落 + 背景雨 + 云朵 |
| 沙尘暴 | 沙粒水平飞行 + 黄褐色蒙版 + 顶底沙尘带 |

#### 天气参数控制
选中天气背景后，面板底部自动展开参数控制区。不同天气显示不同参数：

| 天气 | 可调参数 |
|------|---------|
| 晴天 | 光照强度（0.3-3x） |
| 多云 | 云量（0-100%）、移动速度 |
| 雨天/雷暴 | 云量、降水量（50-500）、下落速度、风力（←5 到 →5）、雷电开关 |
| 雪天 | 降雪量、下落速度、风力、温度（-15°C 到 10°C） |
| 雾天 | 浓度（10-100%） |
| 冰雹 | 云量、冰雹数量、下落速度、风力、雷电开关 |
| 沙尘暴 | 沙尘浓度、风力 |

每个参数都是实时滑块，拖动即时生效。

### 1.4 AI 悬浮球

页面右下角的圆形悬浮按钮：

- **点击展开**：弹出 AI 对话面板
- **快捷入口**（6 个）：帮我规划、找院校、改简历、练面试、找导师、心理支持 — 点击跳转到对应 AI 功能页
- **对话功能**：输入文字，点击发送或按回车，AI 返回模拟回复
- **清空**：清除所有对话历史
- **最小化**：收起面板，保留悬浮球
- **关闭**：完全关闭面板

### 1.5 回到顶部 + 滚动进度

- **滚动进度条**：页面顶部 2px 蓝色渐变条，实时显示阅读进度
- **回到顶部按钮**：滚动超过 300px 后在右下角出现，点击平滑滚动到页面顶部

### 1.6 路由进度条

点击页面内链接切换路由时，页面顶部显示 NProgress 风格的蓝色加载进度条。

### 1.7 Toast 通知

操作反馈弹窗（居中弹出），支持成功/错误/警告/信息 4 种类型。

## 二、首页

路径：`/`

### 2.1 Hero 区域

- **品牌徽章**：显示「全国 X 所高校信息已收录」
- **Slogan**：「研途有我，前路无忧」，「前路无忧」为蓝色渐变文字
- **副标题**：一句话描述平台功能
- **搜索框**：圆角胶囊形状，输入关键词后点击「搜索」按钮或回车，跳转到信息聚合页并带上关键词参数
- **信任标签**：搜索框下方 3 个标签 — 实时更新 / 全国覆盖 / 免费使用
- **热门搜索**：5 个热门关键词（计算机、人工智能、金融、北京大学、清华大学），点击跳转到对应搜索结果

### 2.2 数据统计

4 张统计卡片，数字带 **CountUp 动画**（进入视口时从 0 递增）：
- 收录院校 X 所
- 招生通知 X 条
- 覆盖学院 X 个
- 收录导师 X 位

### 2.3 快速开始（功能入口）

**第一行**（3 列）：保研信息聚合、院校库、导师库 — 点击跳转到对应列表页

**第二行**（3 列，AI 驱动标签）：AI 简历工坊、AI 择校推荐、AI 导师推荐 — 点击跳转，每张卡片有颜色主题标签

**第三行**（4 列）：模拟面试、心理支持、综合规划、保研社群 — 点击跳转

所有卡片 hover 时上浮 + 阴影加深 + 显示「查看详情 →」文字。

### 2.4 最新推免动态

显示最近 4 条推免通知卡片（`InfoCard` 组件），右上角有「查看全部 →」链接跳转到信息聚合页。

### 2.5 热门院校

显示 8 张院校卡片（`SchoolCard` 组件），右上角有「查看全部 →」链接。

### 2.6 功能亮点

6 张亮点卡片，展示核心功能优势。图标 hover 时放大 110%。

### 2.7 用户评价

3 张用户评价卡片，含头像 emoji、姓名、学校、5 星评分、评价内容。Hover 时上浮。

### 2.8 底部 CTA

品牌渐变背景 + 标语 +「立即注册」主按钮 +「了解更多」次按钮。

## 三、保研信息模块

### 3.1 信息聚合 `/info/notices`

#### 页面布局
- 面包屑导航（首页 > 保研信息 > 信息聚合）
- 页面标题 + 图标 + 副标题
- 视图切换按钮（卡片视图 / 紧凑列表视图）
- 筛选面板 + 搜索框
- 通知列表 + 分页

#### 筛选功能（FilterPanel）
| 筛选维度 | 类型 | 选项 |
|---------|------|------|
| 关键词 | 搜索框 | 自由输入 |
| 通知类型 | 下拉 | 全部/夏令营/预推免/宣讲会/入营名单 |
| 进行状态 | 下拉 | 报名中/进行中/未开始/已结束 |
| 学校层次 | 下拉 | 985/211/双一流/其他 |
| 省份 | 下拉 | 34 个省级行政区 |
| 学校 | 下拉 | 31 所热门高校 |
| 学科门类 | 下拉 | 13 个学科 |
| 专业 | 下拉 | 21 个专业 |
| 排序 | 按钮组 | 最新发布/即将截止/热度 |

- 筛选条件变化时自动更新 URL（支持分享筛选链接）
- 已选筛选条件显示为标签，可点击 × 移除
- 「清除全部」按钮重置所有筛选

#### 通知卡片（InfoCard）
每张卡片包含：
- **左侧彩色指示条**（蓝=报名中、绿=进行中、灰=未开始、浅灰=已结束）
- **学校 + 学院**名称（粗体，hover 变蓝）
- **通知标题**（灰色副标题）
- **时间信息**：报名时间（蓝色标签）/ 活动时间（绿色标签）/ 发布日期（兜底）
- **标签行**：夏令营/预推免等类型标签
- **底部统计**：浏览量 + 意向申请人数 + 申请规则
- **右侧状态面板**：状态标签 + 倒计时天数
- **紧急通知**：截止日期 ≤ 3 天时红色边框

点击整张卡片跳转到通知详情页。

#### 分页
底部分页组件：上一页/下一页 + 智能页码省略号 + 当前页高亮。

### 3.2 通知详情 `/info/notices/[id]`

- 面包屑导航
- **顶部信息卡片**：状态标签、时间、类型标签、统计数据
- **侧边栏**（左侧 sticky）：通知概要（类型、学校、学院、目标学位、招生名额、学科方向标签）
- **右侧主内容**：
  - 摘要（带背景色）
  - 申请条件（绿色高亮）
  - 通知原文
  - 来源信息（原文链接、抓取时间）
- **操作按钮**：收藏、分享、官方页面（外部链接）、报名链接
- **右侧卡片**：倒计时 + 往届招生数据

### 3.3 院校库 `/info/schools`

- 搜索框 + 层次筛选（985/211/双一流）+ 省份筛选 + 排序（通知数/名称）
- 院校卡片网格（3-4 列）
- 加载中显示 8 张骨架卡片

#### 院校卡片（SchoolCard）
- **顶部渐变装饰条**（985 蓝 / 211 紫 / 双一流绿）
- **校徽 Logo**（真实校徽图片，加载失败回退到首字母头像）
- **校名 + 层次 Badge**
- **位置信息**（省份 + 城市）
- **统计网格**：学院数 + 通知数（两列灰色底色）
- Hover 时右上角显示箭头图标

### 3.4 院校详情 `/info/schools/[id]`

- 院校基本信息卡片（校徽、校名、层次、位置、官网链接）
- **4 个 Tab 页**：
  - **学院列表**：每个学院显示名称、学科、通知数 Badge、官网链接
  - **通知分布**：水平条形图，按通知数从高到低排列各学院，hover 高亮
  - **最新通知**：该校最新 InfoCard 列表
  - **院校简介**：文字介绍

### 3.5 导师库 `/info/tutors`

- 3 种视图模式切换：
  - **按目标院校**：根据用户设置的目标院校分组
  - **按研究方向**：按学科方向分组，18 种渐变色方案
  - **全部导师**：搜索 + 4 维筛选 + 排序 + 分页

#### 导师卡片（TutorCard）
- 首字母头像 + 姓名 + 职称
- 学校 + 学院
- 招生状态 Badge（绿=招生中 / 灰=未招生）
- 研究方向标签（6 色循环）
- 收藏按钮（心形动画 + 粒子效果）
- 底部统计：地区、论文数、浏览量

### 3.6 导师详情 `/info/tutors/[id]`

- 左侧 sticky 信息卡：头像、姓名、职称、学校、招生状态、联系方式、统计数据、收藏/发邮件按钮、研究方向标签
- 右侧内容区：个人简介、招生要求（绿色高亮）、教育经历、工作经历、代表论文、科研项目、获奖情况、来源信息

### 3.7 竞赛信息 `/info/competitions`

- 搜索框 + 级别筛选（国际/国家/省级）+ 状态筛选 + 学科筛选
- 竞赛卡片列表，每张含：
  - 竞赛名称 + 级别标签 + 状态标签
  - 主办单位
  - 报名/比赛时间、参赛人数
  - 学科标签
  - **含金量星级评分**（1-5 星）
  - **保研加分说明**（橙色高亮区域）
  - 官网外链 + 收藏按钮

### 3.8 期刊会议 `/info/journals`

- **期刊 / 会议 双 Tab 切换**
- 搜索框 + 级别筛选 + 学科筛选
- 列表卡片含：
  - 缩写 + 全名 + 级别标签
  - 影响因子 / 审稿周期 / 录用率
  - **本科生友好度**星级评分
  - 投稿建议提示框
  - 官网外链

## 四、AI 辅导模块

所有 AI 页面共享 Tab 导航布局（毛玻璃效果）。

### 4.1 AI 简历工坊 `/ai/resume`

#### 多简历管理
- 简历列表以卡片展示（名称、更新时间、模块数、模板）
- 创建新简历 / 复制 / 删除
- 切换简历自动保存

#### 8 步分步表单
1. **基本信息**：姓名、性别、民族、出生日期、电话、邮箱、微信、政治面貌、籍贯、地址、个人网站、GitHub、目标院校/专业、照片上传
2. **教育经历**：学校、学院、专业、学位、学制、时间、GPA、排名、核心课程、毕业论文
3. **科研经历**：项目名、类型、角色、导师、实验室、时间、描述（支持 `**加粗**`）、贡献、成果、技术栈
4. **论文成果**：标题、类型、期刊/会议、作者排序、状态、SCI 分区、影响因子、CCF 等级
5. **获奖经历**：名称、级别、等次、颁发机构、类别
6. **实习项目**：单位、行业、职位、类型、描述、成果
7. **技能语言**：技能（类别+内容+熟练度 1-5）、语言（名称+考试+成绩+熟练度）、自我评价
8. **排版设置**：模块管理（拖拽排序、显隐控制）、字体、字号滑块、间距设置、分隔符样式、对齐方式、主题色、照片设置
9. **预览导出**：选择模板 + 实时预览 + PDF 导出

#### 6 套简历模板
| 模板 | 风格 |
|------|------|
| 学术经典 | 蓝色主题，居中姓名，渐变分隔线，单线标题 |
| 现代双栏 | 紫色左侧面板（联系方式+技能进度条），右侧色块标题 |
| 专业商务 | 深灰渐变头部，竖线+横线标题，高密度排版 |
| 优雅学院 | 琥珀色衬线字体，菱形装饰，虚线延伸标题 |
| 极简线条 | 黑白极简，超细灰线，最大留白 |
| LaTeX 学术 | 仿 LaTeX 排版，Georgia 衬线，全大写标题，黑色单线 |

#### AI 建议面板
右侧浮动面板，5 条 Mock 建议（改进/警告/提示），可采纳/忽略。

### 4.2 AI 择校推荐 `/ai/recommend`

#### 输入表单
- 基本信息：本科学校、专业、GPA、排名滑块、期望学位
- 目标方向：学科门类（6 个）+ 具体方向（级联选择）
- 学术背景：科研/论文/竞赛数量计数器 + 英语水平 + 海外经历
- 未来规划：4 种规划卡片 + 导师偏好 + 课题组氛围
- 择校偏好权重：5 个重要性滑块
- 地区偏好：7 大区多选
- 推荐信 / 调剂 / 海外经历开关
- 个人优势/短板自述

#### 结果展示
- 综合评估：总分 + 4 维度环形进度条
- 三档推荐：冲刺（红）/ 稳妥（蓝）/ 保底（绿），共 7 所院校
- 每所院校可展开：推荐理由、往年录取数据、相关导师
- 推荐时间线（5 步保研规划）
- 5 条 AI 建议
- 重新分析 / 导出报告按钮

### 4.3 AI 导师推荐 `/ai/tutor-match`

- 研究方向：学科 + 方向 + 关键词
- 导师偏好：类型、课题组规模、攻读学位
